# Toolkit.Modern.Tests

This project contains unit tests for the ByteForge Toolkit Modern library.

It targets .NET 8.0 and .NET 9.0.

## Folders

- `Unit`: Organized unit tests by category.
  - `CLI`: Tests for command-line interface components.
  - `Configuration`: Tests for configuration parsing and management.
  - `Data`: Tests for data handling, including Audio, CSV, and Database operations.
  - `DataStructures`: Tests for data structures like binary search trees and URLs.
  - `Logging`: Tests for logging functionality.
  - `Mail`: Tests for email handling.
  - `Security`: Tests for security features like AES encryption.
  - `Utils`: Tests for utility classes.

- `TestData`: Sample data files for testing, including a large dummy file and an Access database.
